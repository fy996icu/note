## 拖拽
### React DnD

React DnD 是一组 React 实用程序，可帮助您构建复杂的拖放界面，同时保持组件解耦。它非常适合 Trello 和 Storify 等应用程序，在应用程序的不同部分之间通过拖动传输数据，并且组件会更改其外观和应用程序状态以响应拖放事件。

```bash
# npm
npm i react-dnd react-dnd-html5-backend

# yarn
yarn add react-dnd react-dnd-html5-backend

# pnpm
pnpm add react-dnd react-dnd-html5-backend
```

**插件名称：** [React DnD](https://www.npmjs.com/package/react-dnd)

**在线示例：** [点击查看示例](https://react-dnd.github.io/react-dnd/about)

![img](https://cdn.jsdelivr.net/gh/fy996icu/pics/img/react-dnd.png)
